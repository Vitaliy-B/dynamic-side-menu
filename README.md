# Dynamic Side Menu
Simple Android application with dynamic menu

-------------------------------------

Application loads json from server and displays navigation menu dynamically from its content. 
Page content type: 
 * text 
 * image 
 * web url

Written in Kotlin, uses Retrofit and Glide.
